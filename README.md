Secure Web App
This project is a basic web application demonstrating secure coding practices such as CSRF protection and secure user authentication.

Features:-
1.User signup and login
2.CSRF token implementation
3.Secure data handling with Express.js
4.SQLite database integration

Technologies Used:-
1.HTML, CSS, JavaScript
2.Node.js, Express.js
3.SQLite
